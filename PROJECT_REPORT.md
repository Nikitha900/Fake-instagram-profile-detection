# Fake Instagram Profile Detection

This project identifies and classifies fake Instagram profiles using machine learning models.

## 🔍 Algorithms Used
- Gradient Boosting
- Random Forest
- Logistic Regression
- SVM

## 📊 Accuracy
- Gradient Boosting: 98%
- Random Forest: 96%

## 🛠 Technologies
- Python
- Scikit-learn
- Matplotlib
- Anaconda
- Pycharm
